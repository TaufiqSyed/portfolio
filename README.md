## Getting Started

To run this site locally:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

You can start editing the site by modifying `app/page.tsx`. The page auto-updates as you edit the file.
